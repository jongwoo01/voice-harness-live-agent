# Voice Harness Live Agent

Voice harness live agent for coding workflows with real-time control/approval loops.

Built with Google Gemini Live API for challenge submission.

## Repository Purpose

This repository hosts the public submission baseline for the Google Live Agent Challenge.

## Current Status

- Monorepo implementation is being prepared for initial push.
- Core architecture: local voice/executor + cloud brain + cloud memory.

## Spin-up (Minimum)

```bash
git clone https://github.com/jongwoo01/voice-harness-live-agent.git
cd voice-harness-live-agent
npm install
npm run build
```
